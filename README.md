# Система автоматического распознавания парковочных мест

Разработана для оптимизации использования парковочных пространств в условиях растущей урбанизации и увеличения числа автомобилей.

---

## Актуальность

С ростом числа автомобилей в городах эффективное управление парковочными пространствами становится критически важным. Неорганизованная парковка приводит к:

- **Пробкам**
- **Потере времени водителей**
- **Увеличению выбросов CO₂**

Наша система решает эти проблемы за счёт:

1. **Оптимизации использования парковочных мест** – водители быстрее находят свободные места.
2. **Снижения нагрузки на персонал** – автоматизация исключает ручной контроль.
3. **Интеграции с умными городскими системами** – данные о загруженности парковок помогают анализировать транспортные потоки.

---

## Цель проекта

Разработка нейросетевой системы автоматического распознавания занятых парковочных мест с использованием двух ключевых моделей:

- **U-Net** – для сегментации парковочных зон
- **YOLO** – для детекции транспорта

### Основные задачи:

1. Сегментация парковочных зон с помощью U-Net
2. Детекция транспорта с помощью YOLO
3. Интеграция моделей
4. Разработка интерфейса и тестирование

## Макет системы

1. Система получает данные с камер наблюдения в реальном времени
2. Каждые 5 секунд из видео извлекается ключевой кадр для анализа
3. **U-Net** создаёт маску парковочной зоны, выделяя каждое место
4. **YOLO** детектирует транспортные средства на изображении
5. Система определяет занятость мест, совмещая данные от обеих моделей

> Это позволяет точно и быстро определять, какие места свободны, а какие заняты.

Разработаная система эффективно распознаёт парковочные места и определяет их занятость.
